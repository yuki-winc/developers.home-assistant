家庭助手为家庭控制和家庭自动化提供了一个平台。Home Assistant不仅仅是一个应用程序:它是一个嵌入式系统，提供了与其他消费者现成产品一样的体验:通过一个易于使用的界面来完成入门、配置和更新。

-这[操作系统](操作系统. md)提供运行管理程序和核心的最简单的Linux环境。
-这[监督者](主管. md)管理操作系统。
-这[核心](架构/核心. md)与用户、主管以及物联网设备和服务互动。
